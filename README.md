# YiJing Web App

一個以大衍筮法為核心的周易起卦 Web App。

GitHub Pages:
https://mibsteven.github.io/YiJingWebApp/

## Files

- `index.html` - Web App shell
- `styles.css` - visual design and responsive layout
- `ui.js` - interface interactions
- `yijing-core.js` - casting and reading logic
- `gua-data.json` - 64 hexagram data
- `assets/` - app icon and visual assets
